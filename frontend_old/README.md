# Private AI Agent - Frontend

Next.js frontend for the Private AI Inference Agent dApp.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Configuration

1. Get a WalletConnect Project ID: https://cloud.walletconnect.com
2. Update `config/wagmi.ts` with your project ID

## Features

- Connect wallet (RainbowKit)
- Submit encrypted queries
- View query history
- FHE-powered privacy
